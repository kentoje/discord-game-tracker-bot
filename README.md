# discord-game-tracker-bot

## Installing the project

- clone the project
- install dependencies
with NPM:
```bash
npm i
```
or with Yarn:
```bash
yarn
```

## Running the project in development
- with NPM:
```bash
npm run start
```
- or with Yarn:
```bash
yarn start
```
- run Docker
```bash
docker-compose up -d
```
